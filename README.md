## 📄 How to Convert to Microsoft Word (.docx)

### Option 1: Using Pandoc (Recommended)
1. Install Pandoc from https://pandoc.org/installing.html
2. Open terminal/command prompt
3. Navigate to the folder where you downloaded the files
4. Run:
```bash
pandoc Full_Thesis_Combined.md -o Thesis.docx
```

### Option 2: Google Docs
1. Open Google Docs
2. Paste the content from `Full_Thesis_Combined.md`
3. Go to File → Download → Microsoft Word (.docx)

### Option 3: Online Converter
1. Copy contents of `Full_Thesis_Combined.md`
2. Go to https://pandoc.org/try/
3. Convert Markdown to .docx
